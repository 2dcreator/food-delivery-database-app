# Food Delivery Database App

This is a CSE 412 Phase 03 project implementing a simple Flask web application connected to a PostgreSQL food delivery database.

## Features
- Displays data from customer, restaurant, cuisine, favorite, orders, restaurant_cuisine, and review tables
- Allows adding a new customer
- Allows deleting a customer
- Allows assigning a favorite restaurant to a customer
- Allows removing a favorite restaurant

## Technology Stack
- Python
- Flask
- PostgreSQL
- psycopg2
- HTML
